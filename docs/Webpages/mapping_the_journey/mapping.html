Tracing Civil War Journeys

About this project

This interactive map traces documented and approximate journey stops for soldiers and sailors in the dataset. It is designed as a public-facing research tool: readers can explore movement across enlistment points, transfers, receiving ships, discharge locations, and other recorded stops.

This first version of the map emphasizes documented locations and carefully standardized place names rather than speculative routes. Lines connect each person’s stops in order using stop_order.

How to read the map

  • Lines trace each recorded journey in sequence.
  • Point colors distinguish exact, approximate, and uncertain locations.
  • Line colors distinguish Army and Navy records.
  • Clicking a point opens source-based details for that stop.

Geocode places once and cache them

This chunk geocodes only places that do not already have latitude and longitude. It saves a reusable lookup table so you do not have to geocode everything every time you render.

Explore the map

Use the interactive controls to zoom, pan, and compare Army and Navy journeys. You can turn stops and journey lines on or off in the layer panel.
